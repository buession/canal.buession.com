# buession-canal-core 参考手册


buession canal 核心类库。


---


### 安装

```xml
<dependency>
    <groupId>com.buession.canal</groupId>
    <artifactId>buession-canal-core</artifactId>
    <version>x.x.x</version>
</dependency>
```

该模块定义了 buession canal 核心 API。
